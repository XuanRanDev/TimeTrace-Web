# 时迹使用文档

## 快速开始

### 1. 安装应用

在飞牛 fnOS 应用中心搜索「时迹」，点击安装。或通过命令行安装：

```bash
appcenter-cli install-fpk com.xuanran.archiver.fpk
```

### 2. 启动应用

安装完成后，点击 fnOS 桌面上的时迹图标启动应用。

### 3. 创建第一个扫描任务

1. 打开时迹，进入 **任务管理** 页面
2. 点击 **新建扫描任务**
3. 填写源目录（需要扫描的文件夹路径）
4. 填写目标目录（归档文件存放位置）
5. 选择模式：**复制**（保留源文件）或 **移动**（删除源文件）
6. 点击 **开始扫描**

### 4. 执行归档

扫描完成后，查看待归档文件列表，确认无误后点击 **开始归档**。

## 功能详解

### 智能扫描

时迹会递归扫描源目录下的所有文件，不区分文件类型。扫描过程中会：

1. **提取时间信息**：优先读取 EXIF 拍摄日期，其次识别文件名中的时间戳（支持 13 位 Unix 毫秒），最后使用文件修改时间
2. **计算 SHA1 哈希**：用于去重和后续的重复文件查找
3. **增量扫描**：已处理过的文件会被跳过，仅处理新增和变更的文件

### 分类规则

分类规则决定了文件归档到哪个子文件夹。规则支持以下匹配方式：

| 匹配方式 | 说明 | 示例 |
|---------|------|------|
| 扩展名 | 按文件扩展名匹配 | `.jpg`, `.mp4` |
| 文件名 | 按文件名模式匹配（支持通配符） | `*微信*`, `IMG_*` |
| 源文件夹 | 按源文件所在目录匹配 | `/DCIM/`, `/WeChat/` |
| EXIF 品牌 | 按拍摄设备品牌匹配 | `Apple`, `Canon` |
| EXIF 型号 | 按拍摄设备型号匹配 | `iPhone 15 Pro` |

规则支持优先级排序，排在前面的规则优先匹配。

### 命名规则

命名规则支持模板化重命名，可用变量：

| 变量 | 说明 | 示例值 |
|------|------|--------|
| `{date}` | 文件日期（YYYY-MM-DD） | `2024-06-15` |
| `{year}` | 年份 | `2024` |
| `{month}` | 月份 | `06` |
| `{day}` | 日期 | `15` |
| `{sha1}` | 文件 SHA1 哈希（前8位） | `a1b2c3d4` |
| `{ext}` | 文件扩展名 | `.jpg` |
| `{name}` | 原始文件名（不含扩展名） | `IMG_001` |

**示例**：`{date}_{sha1}{ext}` → `2024-06-15_a1b2c3d4.jpg`

### 定时任务

支持基于 Cron 表达式的定时调度，可配置自动执行扫描和归档。

**Cron 表达式格式**：

```
分 时 日 月 周
```

**常用示例**：

| 表达式 | 说明 |
|--------|------|
| `0 2 * * *` | 每天凌晨 2 点执行 |
| `0 */6 * * *` | 每 6 小时执行一次 |
| `0 3 * * 1` | 每周一凌晨 3 点执行 |
| `0 0 1 * *` | 每月 1 日零点执行 |

### 重复文件查找

基于 SHA1 哈希查找重复文件。查找结果会按哈希值分组，方便对比和清理。

## 归档目录结构

归档后的文件按以下结构组织：

```
目标目录/
├── 2024/
│   ├── 01/
│   │   ├── Photos/        # 图片
│   │   ├── Videos/        # 视频
│   │   ├── Music/         # 音频
│   │   ├── Documents/     # 文档
│   │   ├── Archives/      # 压缩包
│   │   └── Others/        # 其他
│   ├── 02/
│   │   └── ...
│   └── ...
└── 2025/
    └── ...
```

## 常见问题

### Q: 扫描速度很慢？

A: 扫描速度受磁盘 I/O 和文件数量影响。首次扫描大量文件时会较慢，后续增量扫描会快很多。

### Q: 如何修改已归档文件的位置？

A: 在文件浏览页面可以查看归档结果，手动移动文件即可。

### Q: 支持哪些文件格式？

A: 时迹支持所有文件格式，不区分类型。时间提取支持 JPEG、TIFF、HEIC 等常见图片格式的 EXIF 信息。

### Q: 数据存储在哪里？

A: 所有数据存储在 fnOS 的应用数据目录中，使用 SQLite 数据库，无需额外配置。
