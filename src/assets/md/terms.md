## 二、应用说明

时迹是一款运行于飞牛 fnOS 之上的智能文件归档工具，旨在将散落于各处的文件自动整理为 `yyyy/MM/Photos` 结构，便于用户按月进行备份与增量备份。其主要功能包括：

- 递归扫描用户指定目录下的全部文件；
- 提取 EXIF 元数据及文件名中蕴含的时间戳信息；
- 按年、月、类型自动归档文件至指定目录；
- 基于 SHA1 哈希值进行文件去重；
- 支持自定义分类规则与命名规则；
- 支持定时自动执行扫描与归档任务。

## 五、隐私政策

### 5.1 数据收集原则

本应用遵循最小数据收集原则。本应用的核心功能仅在用户设备本地运行，不会主动收集、上传、传输或存储用户的个人身份数据至任何外部服务器或第三方服务。**唯一例外**为会员服务相关通信，具体说明见[第 5.4 条](#54-网络通信)。

### 5.2 本地数据存储

以下数据仅存储于用户设备本地的 SQLite 数据库中，由用户自行管理：

- **扫描记录**：文件路径、文件大小、SHA1 哈希值及元数据；
- **文件索引**：源文件目录索引及其变更记录；
- **操作日志**：扫描、归档等操作的执行记录；
- **任务配置**：定时任务、分类规则与命名规则；
- **用户设置**：端口配置、目录授权等本地设置。

### 5.3 用户认证

本应用通过飞牛 fnOS 统一网关进行用户身份识别，具体方式为读取请求头中的用户 ID 与管理员标识。用户信息仅用于本地权限控制与数据隔离，不会被传输至任何外部服务器。

### 5.4 网络通信

本应用仅在以下情形下产生网络通信：

- 响应用户对 Web 界面发出的请求（本地 HTTP 服务）；
- 与飞牛 fnOS 网关之间的反向代理通信；
- 当用户访问会员授权页面时，本应用会向开发者服务器发送请求，以查询当前会员套餐及价格信息；当用户选择开通会员时，本应用将向开发者服务器提交包含**匿名**设备唯一标识符的授权请求，用于会员资格的激活与核验。上述全部通信均不包含用户的个人信息。

### 5.5 第三方组件

本应用不集成任何第三方分析工具、广告 SDK、数据收集服务或追踪组件。

### 5.6 数据删除

用户卸载本应用时，本地数据库及配置文件可能不会被自动删除，具体取决于飞牛 fnOS 的卸载机制。用户可手动删除应用数据目录，以彻底清除全部本地数据。
