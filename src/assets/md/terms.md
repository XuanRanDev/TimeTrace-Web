## 二、应用说明

时迹是一款运行在飞牛 fnOS 上的智能文件归档工具，把散落的文件自动整理成 `yyyy/MM/Photos` 结构，方便按月备份和增量备份。主要功能包括：

- 递归扫描用户指定目录下的所有文件
- 提取 EXIF 元数据和文件名中的时间戳信息
- 按年/月/类型自动归档文件到指定目录
- 基于 SHA1 哈希的文件去重
- 自定义分类规则和命名规则
- 定时自动扫描和归档任务

## 五、隐私政策

### 5.1 数据收集原则

本应用遵循最小数据收集原则。本应用的核心功能仅在用户设备本地运行，不会主动收集、上传、传输或存储用户的个人身份数据至任何外部服务器或第三方服务。**唯一例外**为会员服务相关通信，具体说明见第 5.4 条。

### 5.2 本地数据存储

以下数据仅存储在用户设备本地的 SQLite 数据库中，由用户自行管理：

- **扫描记录**：文件路径、大小、SHA1、元数据
- **文件索引**：源文件目录索引和变更记录
- **操作日志**：扫描、归档等操作记录
- **任务配置**：定时任务、分类规则、命名规则
- **用户设置**：端口、目录授权等配置

### 5.3 用户认证

本应用通过飞牛 fnOS 统一网关进行用户身份识别（读取请求头中的用户 ID 和管理员标识）。用户信息仅用于本地权限控制和数据隔离，不会被传输到任何外部服务器。

### 5.4 网络通信

本应用仅在以下情况产生网络通信：

- 响应用户的 Web 界面请求（本地 HTTP 服务）
- 飞牛 fnOS 网关的反向代理通信
- 当用户访问会员授权页面时，本应用会向开发者服务器发送请求以查询当前会员套餐及价格信息；当用户选择开通会员时，本应用将向开发者服务器提交包含**匿名**设备唯一标识符的授权请求，用于会员资格的激活与查询验证。上述通信均不包含用户的个人信息。

### 5.5 第三方组件

本应用不集成任何第三方分析工具、广告 SDK、数据收集服务或追踪组件。

### 5.6 数据删除

用户卸载本应用时，本地数据库和配置文件可能不会被自动删除（取决于飞牛 fnOS 的卸载行为）。用户可手动删除应用数据目录以彻底清除所有本地数据。
